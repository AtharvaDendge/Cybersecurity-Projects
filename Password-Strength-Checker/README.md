# 🔐 Password Strength Checker (GUI)

A Python-based GUI tool that evaluates password strength in real-time and provides tips for creating stronger passwords. Built with Tkinter.

## Features
- Real-time password strength evaluation  
- Strength categories: Very Weak → Very Strong  
- Show/hide password option  
- Visual feedback with color coding  
- Tips for creating strong passwords  

## Usage
1. Install Python 3.x  
2. Run the script:
   ```
   python password_strength_checker.py
   ```
3. Enter a password and view the strength and tips in the GUI.

## Requirements

- Python 3.x
- Tkinter (usually included with Python)

## Why Use This Tool?

- Learn good password hygiene
- Test your passwords before using them in accounts
- Educational tool for cybersecurity beginners
